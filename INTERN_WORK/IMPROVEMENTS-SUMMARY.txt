## 🎓 Những điều học được

### Feature 1 — Toast Notification
- **Context API** — chia sẻ state (toast) cho toàn app
- **useCallback** — tránh re-render không cần thiết
- **setTimeout** — tự động xóa toast sau X giây
- **CSS animation** — hiệu ứng slide-in từ bên phải

### Feature 2 — Confirmation Dialog
- **Client Component** — xử lý interaction (click, state)
- **Modal/Overlay pattern** — fixed position, z-index cao
- **Conditional rendering** — hiện/ẩn dialog theo state
- **Optimistic UI** — xóa ngay rồi gọi API

### Feature 3 — Skeleton Loading
- **Suspense** — React component xử lý loading state
- **Skeleton UI pattern** — placeholder có animation
- **CSS animation pulse** — hiệu ứng loading nhẹ nhàng
- **Tailwind animate-pulse** — class có sẵn
