# Intern Community Hub

Intern Community Hub is a web-based platform that enables developers to share, discover, and evaluate mini-application modules contributed by the community.

The platform allows users to browse a curated list of approved modules, submit their own projects for review, and interact with existing modules through a voting system. Submissions go through a moderation process where administrators review and either approve or reject them before they become publicly visible.

In addition to being a functional application, this repository also serves as an open-source internship challenge. Instead of traditional assessments, candidates are encouraged to demonstrate their technical skills by contributing directly to a real-world codebase through pull requests.

This approach allows evaluation of not only coding ability, but also code quality, problem-solving, collaboration, and the effective use of modern development tools.
